# Estudos da Linguagem TypeScript

Estudos sobre a linguagem TypeScript

### Conteúdo

- Annotation
- Inference
- Type
- Union Type
- Array
- Any

### Considerações

- Instalar typescript globalmente e para desenvolvimento
  `npm i typescript -G -D`
- Criar tsconfig
  `tsc --init`
- Para compilar automaticamente (watch)
  `tsc -w`
